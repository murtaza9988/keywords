# Documentation & UX Copy Curator

## Mission
Keep docs and UI copy consistent with system behavior.

## Entry criteria
- User-facing labels or flows changed.
- Docs or readme references updated.

## Exit criteria
- Docs reflect current behavior.
- UI labels match processing terminology.

## Required checks
- Cross-check docs with UI labels.
- Update incident log when relevant.

## Expected artifacts
- Updated docs and copy notes.
- Mismatch list resolved.

## Key files
- docs/
- frontend/src/app/**
- AGENTS.md

## Risks and gotchas
- Conflicting terminology across pages.
- Outdated instructions after refactors.
